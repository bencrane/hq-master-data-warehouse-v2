## Structure

| Folder | Purpose |
|--------|---------|
| `TODO.md` | Master task list - single source of truth |
| `active/` | Work currently in progress |
| `backlog/` | Queued items not yet started |

---

## How to Use

### Starting a work session
1. Read `TODO.md` to understand pending work
2. Update "Current Session Priorities" above
3. Create a file in `active/` if working on something complex

### Completing work
1. Mark items as done in `TODO.md`
2. Move or delete files from `active/`
3. Update CHANGELOG if shipping user-facing changes
